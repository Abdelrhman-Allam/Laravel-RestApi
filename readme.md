### How to start 
1. Clone the project:

    git clone https://github.com/Abdelrhman-Allam/Laravel-RestApi.git

    php artidan generate:key
    
2. configur datebase on .env file
3.run migrations to create tables

   php artisan migrate
   
4.list routes

5. start Postman
